# Contributing

Thanks for taking the time to report a bug or suggest an improvement.

## Bug Reports

Please include:

- REAPER version
- operating system
- exact steps to reproduce
- expected result
- actual result
- whether the problem happens with a small test project
- screenshots or a short screen recording if that helps

If possible, include a minimal example with:

- item positions
- region positions
- what fragments you expected to keep

## Feature Requests

Please describe:

- the problem you are trying to solve
- the current workaround, if any
- the behavior you would like to see

## Pull Requests

- Keep changes focused and easy to review.
- Do not add dependencies unless they are clearly necessary.
- Preserve stock REAPER compatibility.
- Run the Lua syntax check before submitting changes.
