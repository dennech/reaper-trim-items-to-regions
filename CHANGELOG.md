# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-03

### Added

- Initial public release of the REAPER Lua ReaScript.
- Region-aware trimming for selected media items without moving remaining fragments.
- Safe no-op behavior when no items are selected or when the project has no regions.
- ReaPack packaging metadata and repository index support.
- GitHub issue templates and CI validation for Lua syntax and ReaPack index generation.
